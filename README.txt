PROJECT OVERVIEW
----------------
This is a fully functional coffee shop website with e-commerce capabilities,
user authentication, admin panel, and customer management features.

WHAT'S INCLUDED
---------------
1. Custom Logo (logo.svg)
   - Professional SVG design
   - Coffee cup illustration
   - Integrated throughout the website

2. Login & Authentication System
   - login.html - Secure login page
   - register.html - User registration
   - profile.html - User profile management
   - auth.js - Authentication logic
   - Session management with localStorage

3. Website Pages
   - index.html - Homepage with hero section
   - features.html - Coffee shop features
   - products.html - Product catalog with shopping cart
   - contact.html - Contact form
   - customers.html - Customer management (admin)
   - messages.html - Message inbox (admin)
   - admin.html - Admin dashboard

4. User Manual
   - USER_MANUAL.pdf - Complete PDF manual with screenshots
   - USER_MANUAL.html - HTML version
   - USER_MANUAL.md - Markdown version
   - screenshots/ - All page screenshots

5. Assets
   - style.css - Main stylesheet
   - script.js - Main JavaScript
   - images/ - Product images and graphics

HOW TO RUN
----------
1. Extract all files to a directory
2. Open a terminal in the directory
3. Run: python3 -m http.server 8000
4. Open browser to: http://localhost:8000
5. Navigate to index.html
